# Shim Server

> Healing Music Player App Server

## Installation

- Clone the repo:

```bash
git clone https://github.com/prography/shim-server.git
```

## Build Setup

```bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run start
```
